1. close "ComfyUI"
2. copy the ComfyUI_windows_portable folder to your "ComfyUI" so that the directories match and the files are overwritten. Don't forget to update your "ComfyUI"
3. run ComfyUI_windows_portable\update\llava_install.bat to install the necessary libraries
4. start "ComfyUI"

the code for the comfyUI node was used from this project - https://github.com/natlamir/LLaVA-Windows
